# 前端工程化定义与解决的问题
  - 前端工程化定义：
  - 解决的问题：
    + 传统语言或语法的弊端；
    + 无法使用模块化/组件化
    + 重复的机械式工作
    + 代码风格统一，质量保证
    + 依赖后端服务接口支持
    + 整体依赖后端项目
  - 工程化的表现：一切以提高效率、降低成本、质量保证为目的的手段都属于工程化、一切重复性的工作都应该被工程化
  - 工程化 != 工具
